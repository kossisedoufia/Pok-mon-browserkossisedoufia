# Vite-Tailwind-JS Template

This is a template for a Vite project using Tailwind CSS and JavaScript. It includes the necessary configuration and setup to get you started quickly with these technologies.

## Features

- Vite for fast development and building
- Tailwind CSS for utility-first styling

## Getting Started

After cloning the repository, you can install the dependencies and start the development server:

```bash
bun install
bun run dev
```

This will start the Vite development server, and you can view your application in the browser at `http://localhost:5173`.
